> ⚡ A modern Discord bot for the **VoidCraft community** — built to **link accounts, track invites, reward roles, manage tickets, and provide analytics**.  
> Crafted with ❤️ using **Node.js**, **Discord.js**, and **MongoDB**.

## ✨ Features
- 🔗 **Account Linking** – Connect external accounts with Discord profiles.
- 📊 **Invite Tracking** – Monitor server invites and generate leaderboards.
- 🎖️ **Role Rewards** – Automatically assign roles based on activity or linked accounts.
- 🎟️ **Ticket System** – Handle support requests with ease.
- 📈 **Server Analytics** – Gain insights into member activity and growth.
- ⚡ **Rate Limiting** – Prevent spam and abuse with smart controls.
